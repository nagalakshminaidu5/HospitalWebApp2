 Hospital Management System (Thymeleaf)

##  Project Overview

The **Hospital Management System** is a web-based application developed using **Spring Boot** and **Thymeleaf**.
It helps manage hospital operations such as patient records, doctor details, and appointment scheduling.

---

## Features

*  Doctor Management (Add, View, Update, Delete)
*  Patient Management
*  Appointment Booking System
*  Appointment Approval/Denial
*  Basic Validation & Error Handling
*  User-friendly UI with Thymeleaf

---

##  Technologies Used

* Java
* Spring Boot
* Thymeleaf
* Spring MVC
* Spring Data JPA
* MySQL (or any relational database)
* HTML, CSS

---

src/

---

##  How to Run the Project

1. Clone the repository:

```
git clone https://github.com/nagalakshminaidu5/HospitalWebApp2.git
```

2. Open in IDE (Eclipse / IntelliJ)

3. Configure Database:

* Update `application.properties` with DB credentials

4. Run the application:

```
Run as Spring Boot App
```

5. Open browser:

```
http://localhost:8080
```
